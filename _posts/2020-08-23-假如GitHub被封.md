2020 年还未结束, 2020 年以后依然也「什么事情都可能发生」。

但是假如 GitHub 被封了呢?

那我所有的 GitHub Pages 都访问不了了, 所有的代码仓库都访问不了了, Raku 最新的东西也没法看了, 那我这个民科程序员还写啥子代码呦, 去转行说脱口秀麽(删)?

也许只能看特效酷炫、衣服华丽、眼神冷漠、所有人都在谈恋爱的国产电视剧打发时间了。

那时候, 这个 9,600,000 平方公里的世界就更无聊了, 只能去北极偷回我的源代码了。
